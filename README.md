# 🪑 Furniture Shop E-Commerce Website

A full-stack e-commerce website developed for a furniture store using Django framework for backend. The platform allows users to browse furniture products, manage carts, and experience a smooth online shopping interface.

## 🚀 Features

- Responsive furniture product showcase  
- Category-wise product browsing  
- Shopping cart functionality  
- Secure backend using Django framework  
- Admin panel for product management  
- User-friendly navigation  
- Mobile-responsive design  
- Basic checkout flow  

## 💻 Technologies Used

### Frontend:
- HTML
- CSS
- JavaScript

### Backend:
- Django (Python)
- SQLite / Database (MySQL)

## 🔐 Admin Features

- Add/Edit/Delete products  
- Manage categories  
- Control product details from Django admin panel  

## 🎯 Purpose

This project was developed as a real-world furniture e-commerce solution to improve full-stack web development skills, backend integration, and client-oriented project deployment.
